# We are under attack!

本仓库用于记录 B3log 系列站点被攻击的记录，不定期更新，欢迎 Fork！

## 使用

使用你服务器的公网 IP，通过 GitHub 提供的搜索功能搜索本库（“In this repository”），从搜索命中结果来判断你的服务器是否已经沦为攻击者的肉鸡。

## 来源

原始日志数据来自 B3log 正在使用的一些服务：

* 七牛云融合 CDN 日志
* 阿里云 DDoS 日志
* 应用服务器日志
