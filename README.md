# We are under attack

本仓库用于记录 B3log 系列站点被攻击的记录，不定期更新。

欢迎 Fork！

## 意义

* 检查自己的服务器是否已经沦为攻击者的肉鸡
* 让攻击者“杀敌 1000，自损 800”
* 最终，希望大家的网站都能免遭攻击，稳定运行

## 使用

通过 GitHub 提供的搜索功能，使用自己服务器的公网 IP 搜索本库（“In this repository”）。

## 来源

原始日志数据来自 B3log 正在使用的一些服务：

* 七牛云融合 CDN 日志
* 阿里云 DDoS 日志
* 应用服务器日志
